## Learniverse Application

Learniverse is an Android-based learning platform designed to provide students with an interactive and modern educational experience.
The application includes study materials, quizzes, coding programs, chatbot support, and user-friendly learning features.

## Features

-  User Authentication System
-  Courses and Topics Management
-  Chatbot for Query Assistance
-  Quiz/Test Series System
-  Coding Programs and Exercises
-  Result and Performance Tracking
-  Feedback and Rating System
-  Professional Black & White UI Design
-  API Integration with Node.js Backend
-  MySQL Database Connectivity

## Technologies Used

Frontend

- Kotlin
- XML
- Android Studio

Backend

- Python Flask

Database

- MySQL

Other Tools

- Retrofit API
- Git & GitHub

## Installation Steps

1. Clone Repository
2. Open in Android Studio

- Open Android Studio
- Click Open Existing Project
- Select the Learniverse folder

3. Backend Setup
4. Database Setup

- Open MySQL
- Import:
learniverse.sql

5. Configure database credentials inside backend configuration files.
